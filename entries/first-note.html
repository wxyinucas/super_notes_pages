第一篇示例 ​

这篇文件展示公式、旁注和分栏的基本写法。正文仍然是普通 Markdown，所以你可以像写 LaTeX 小节一样，把内容按主题拆到不同文件。

旁注

这里支持 Markdown 和 KaTeX：a2+b2=c2a^2+b^2=c^2a2+b2=c2。宽屏时它会浮到正文右侧，小屏时回到正文流里。

一个公式段落 ​

考虑一个最小二乘目标：

L(θ)=∑i=1n(yi−xi⊤θ)2L(\theta)=\sum_{i=1}^{n}(y_i - x_i^\top\theta)^2 L(θ)=i=1∑n​(yi​−xi⊤​θ)2

当你想补充上下文，但又不想打断主线时，就可以放一个旁注。

也可以用 slot 写法。这里的 Markdown 会先由 VitePress 渲染，因此适合多段旁注。

分栏片段 ​

左栏可以放定义、摘要或引用。这个组件只负责排版，不改变 Markdown 的语义。

右栏可以放推导、对照笔记或 TODO。移动端会自动回到单栏。

后续可以扩展 ​

  • 自动从 entries/ 生成日志索引。
  • 增加标签、日期、草稿状态。
  • 为长文增加更接近 LaTeX 的 include 约定。
